## General Info

Particle plugin that spawns particle when players move!

## Commands

```/pa disable/enable``` - Enable or disable particle<br>
```/pa set <id>``` - Sets particle (FOR OP's ONLY!!)<br>
```/pa list``` - List available particles to set<br>
